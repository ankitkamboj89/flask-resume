This Flask web app predicts whether a telecom customer is likely to churn based on inputs like tenure, contract type, internet services, and monthly charges. The model was trained on the Telco Customer Churn dataset using a Random Forest Classifier with Label Encoding and StandardScaler preprocessing.

The pipeline includes:
  • Data cleaning and type conversions (handling TotalCharges as float)
  • Label encoding for all categorical features
  • Train-test split (80/20) with standardization of numeric fields
  • RandomForest training with saved model, scaler, and label encoders using joblib
Evaluation Metrics:
F1 Score (Churn = Yes): ~0.63
F1 Score (Churn = No): ~0.86
Overall Accuracy: ~79%
